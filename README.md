# discount-game

A web application that shows game deals from various stores.

## Features

* Fetches and displays game deals from the CheapShark API.
* Uses Firebase Hosting for deployment.
* Implements a serverless function using Cloud Functions for Firebase to fetch data from the API.

## Deployment

This project is deployed using Firebase Hosting. The live version can be accessed at [https://your-firebase-project-id.web.app](https://your-firebase-project-id.web.app).
